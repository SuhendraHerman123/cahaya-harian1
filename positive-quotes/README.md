# 🌟 Cahaya Harian — Daily Positive Quotes

Aplikasi web **daily positive quotes** dengan gambar indah, tema motivasi kehidupan & stress relief.

## ✨ Fitur
- 📅 **Quote Harian** — quote berbeda setiap hari otomatis
- 🖼️ **Gambar Relevan** — foto dari Unsplash yang sesuai tema quote
- ❤️ **Favorit** — simpan quote favoritmu (tersimpan di browser)
- 🔀 **Jelajahi** — browse semua 12 quote
- 📤 **Bagikan** — share ke WhatsApp, media sosial, dll
- 🎨 **Dark UI** premium dengan animasi halus
- 📱 **Responsive** — tampil cantik di HP dan desktop

## 🚀 Deploy ke Vercel (3 langkah)

### Langkah 1: Upload ke GitHub
1. Buka [github.com](https://github.com) → **New repository**
2. Nama repo: `cahaya-harian`
3. Klik **Create repository**
4. Upload semua file dari folder ini ke repo tersebut

### Langkah 2: Deploy ke Vercel
1. Buka [vercel.com](https://vercel.com) → Login dengan GitHub
2. Klik **"Add New Project"**
3. Pilih repo `cahaya-harian`
4. Klik **"Deploy"** (tidak perlu setting apapun!)

### Langkah 3: Selesai! 🎉
Vercel akan otomatis build dan deploy. URL aplikasi kamu akan seperti:
`https://cahaya-harian.vercel.app`

## 💻 Jalankan Lokal

```bash
npm install
npm run dev
```
Buka http://localhost:3000

## 🛠️ Tech Stack
- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Unsplash** (gambar gratis)
